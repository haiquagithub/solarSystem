# solarSystem
Solar System Animation is a web project that visually simulates the motion of planets around the Sun using pure HTML and CSS. It demonstrates how CSS @keyframes, transform, and animation properties can be combined to create smooth orbital rotations and dynamic celestial effects without JavaScript.
# Solar System Animation

## 📖 Overview
This project showcases a creative animation of the **Solar System**, where planets revolve around the Sun in circular orbits.  
It is built entirely with **HTML and CSS**, highlighting the power of CSS animations to simulate real-world motion.

## ✨ Features
- ☀️ Sun fixed at the center with glowing effect
- 🌍 Planets revolving around the Sun
- 🌙 Optional moons orbiting planets
- 🎨 Pure CSS animations (no JavaScript required)
- 📱 Responsive design for different screen sizes

## 🛠️ Technologies Used
- **HTML5** – for structure
- **CSS3** – for styling and animations (`@keyframes`, `transform`, `transition`)

## 🚀 How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Run the `index.html` file in any modern browser.
4. Watch the animated Solar System in action.
